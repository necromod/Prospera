# Prospera


Prospera, um sistema de administração Financeira Pessoal,é uma plataforma web para a administração da vida financeira de pessoas físicas, do homem comum. Nossa missão principal é entregar uma interface de fácil acessibilidade para o cliente final ter controle total de sua vida financeira, a gerenciando com poucos cliques e obtendo relatórios de desempenho de economia e gastos, tudo isso em um sistema automatizado e autônomo.

O maior diferencial dessa plataforma é seu design pensado principalmente na acessibilidade, trazendo para o usuário final jovem, adulto ou idoso uma interface simples e intuitiva, em um uso simplificado para todos os públicos, seja para ter um controle de toda a vida financeira ou uma pequena administração gastos e planejamentos com orçamentos.

Permitindo os usuários ingressem suas informações financeiras e transações, Prospera busca automatizar todas as ferramentas de plataforma, podendo realizar ou receber transações e ter um extrato atualizado em tempo real com cada movimentação, tal como histórico de movimentações com cada pessoa já relacionada, e também atualizando saldo de contas bancárias cadastradas. 
Através de um consultor automatizado, os usuários receberão análises detalhadas, estatísticas e gráficos personalizados para avaliar seus gastos, metas financeiras e planos de economia. Além disso, implementaremos um sistema de planejamento financeiro que mostra o estado financeiro atual e futuro, mantido sempre atualizado pelas próprias movimentações da conta do usuário.
